Объект - Шариковая ручка.
  Состояние :
   - кол-во чернил.
  Поведение :
   - писать (при использовании этого поведения объект сам уменьшается кол-во чернил)
   
Объект - Чайник.
  Состояние :
   - вкл. выкл.
   - кол-во воды
   - температура
  Поведение :
   - включить - При включении начинает повышаться температура.
   - выключить - При выключении температура начинает снижаться.
   
Объект - Ноутбук.
  Состояние :
   - вкл. выкл.
   - уровень заряда батареии.
  Поведение :
   - включение - При включении ноутбук переходит из состаяния выкл в вкл.
   - выключение - При выключении ноубук переходит из состаяния вкл в выкл.
   - подзарядить - Используя это поведение вы повышаете уровень заряда батареи.
   
Объект - Книга
 	Состояние:
    - кол-во страниц.
    - открыта закрыта.
    - автор.
  поведение:
    - перелистнуть на следующаю страницу срабатывает выражение (n+1), где n - страница на которой открыта книга.
    - открыть - При ипользовании этого поведения, объект книга перейдёт с состояния закрыта в состаяние открыта.

Объект - Телефон
	Состояние:              
    - разряженный.
    - заблокированный/разблокированный.
  Поведение:
    - заряжать (меняется уровень заряда).
    - разблокировать.
    - заблокировать.

Объект - Часы
  Состояние:

Объект - Монитор
  Состояние:
    - вкл. выкл.
    - яркость.
    - контрастность.
  Поведение:
    - включить - Используя это поведение вы меняете состояние выкл на вкл.
    - повышение яркости - Используя это поведение вы повышаете кол-во яркости на n .

Объект - Горшок с цветком
	 Состояние:   
    - полит. не полит.
    - кол-во воды 
    - стоит на подоконнике.
	  Поведение:
    - поливать - При использовании этого поведения, повышается кол-во воды .
