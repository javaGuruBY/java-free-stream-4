*Включить ноутбук
1. достать ноутбук из чеха
2. поставить / положить его
3. открыть 
4. нажать на кнопку включения

*открыть окно
1. подойти к окну
2. повернуть ручку
3. потянуть ручку на себя

*положить тетрадь в портфель
1. взять портфель
2. открыть его
3. положить тетрадь в портфель
4. закрыть портфель

*попить воды из бутылки
1. взять бутылку
2. поворачивать крышку против часовой стрелки, пока бутылка не откроется
3. вывпить воды
4. закрыть бутылку (крутить крышку по часовой стрелке)

*зайти в вк через телефон
1. достать телефон
2. разблокировать его
3. найти приложение ВК
4. нажать на него
