Объект - Домашний питомец.
Состояние :
           Сытый/Голодный.
           Спит/Бодорствует.
Поведение :
           Накормить - сытый. 
           Разбудить - бодурствует. 
           Не кормить - Произносит с особой интонацией "мяу" в 3 часа ночи.
           
Объект - Телефон.
Состояние :
           Заряд батареи (%).
           Вкл./Выкл.
           Режим звука: вибрация/без звука/звук.           
Поведение :
           Юзать в ютубе - Разряжаться, вставить зарядку - заряжается.
           Нажать кнопку включения/выключения - Включаться/выключаться.
           сменить режим звука - Вибрировать, Издавать звуки (звенеть).
           
Объект - Ноутбук
Состояние :
           Яркость экрана.
           Заряд батареи (%).
           Громкоть звука
Поведение :   
           Увеличивать/уменьшать яркость.
           Заряжаться/Разряжаться.
           Увеличивать/Уменьшать громкоть звука.
           
Объект - Стул
Состояние :
           Устойчивый/Неустойчивый.
           Загрязнения (чистый/грязный).
Поведение :
           Развалиться/Сохранить текущее состояние под влиянием внешнего воздействия.
           Загрязьняться под влиянием внешнего воздействия
          
Объект - Наручные электроные часы 
Состояние :
           Заброкирован/Разблокирован дисплей.
           Заряд батареи (%).
Поведение :
           Не Нажать - Не отображать текущее сотсояние даты/времени на экране дисплея (Неактивный дисплей при блокировке экрана).
           Нажать - Отображает дату, время, количесво шагов и т.д. (Активный дисплей при разблокировке экрана).
           установить в зарядное устройство - Заряжаться/ использовать по назанчению - Разряжаться.

Объект - Верхняя одежда
Состояние : 
           Уровень тепла в верхней одежды (Осенняя/весенняя или зимняя)
           Водонепроницаемость.
Поведение :
           Одеть - Согревать/не греть в холодную погоду.
           Стоять под дождем - Пропускать/не пропускать влагу в влажную погоду.
           
Объект - Стиральная машина
Состояние :
           Вкл./Выкл.
           Режим стирки.
           Температура стирки.
           Колебания/Покой.
Поведение :
           Включить - стирает/ не включить - стоять не подвижно.
           повысить температуру в режиме стирки - повышается температура тена в стиральной машине.
           включить отжим - Изменяется состояние покоя (вращение барабана) при  стирке.

Объект - Холодильник.
Состояние :
           Вкл. выкл. 
           Температура
Поведение :
           включить - При включении начинает понижаться температура до определенной отметки.
           выключить - При выключении температура начинает повышаться до температуры окружаюещей среды.
           
Объект - Фломастер.
Состояние :
           кол-во чернил.
Поведение :
           писать (при использовании этого поведения объект сам уменьшается кол-во чернил).
     
Объект - Планшет.
Состояние -
           Текущие приложение.
           Уровень звука.
Поведение -
            повысить/понизить уровень звука - увеличивается/ понижается громкость звука при нажатии.
            перейти на определенное приложение.
