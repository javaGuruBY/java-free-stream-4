Объект - Стикер.
        Состояние:
        бумага клейкая
        бумага для записей
        Поведение:
        наклеить стикер на что-либо (приэтом количество стикеров в стопке уменьшается)
        написать заметку

        Объект - Power-bank.
        Состояние:
        включен/выключен
        заряд аккумулятора
        Поведение:
        зарядка аккумулятора телефона (уменьшается колчество заряда)
        зарядка аккумулятора Power-bank (увиличевается колчество заряда Power-bank)

        Объект - Степлер.
        Состояние:
        количество скоб
        Поведение:
        сшивать бумагу (уменьшение количества скоб в степлере)

        Объект - Магнитола.
        Состояние:
        вкл/выкл
        проигрывание треков
        радио
        Поведение:
        перемотка треков (переключение в плейлисте)
        настройка волны радио

        Объект - Сковорода.
        Состояние:
        Холодная
        Горячая
        Поведение:
        Греется при контакте с электро плитой

        Объект - Мыло.
        Состояние:
        жидкое
        твердое
        Поведение:
        налить на руки (приэтом количество уменьшается)
        мыть мылом руки (объем мыла уменьшается)

        Объект - шарик.
        Состояние:
        резиновый
        сдутый
        Поведение:
        растянуть
        наполнить воздухом при этом он станет круглый

        Объект - ноутбук.
        Состояние:
        открыт/закрыт
        Поведение:
        проверить электронную почту - при этом ноутбук открыт

        Объект - земля.
        Состояние:
        сухая
        Поведение:
        впитывает воду


        Объект - стакан.
        Состояние:
        пустой
        Поведение:
        наполнить водой
